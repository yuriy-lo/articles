# Articles & Projects

This repository contains example code for published articles that explore programming concepts and techniques.

## Projects

### [args-reflect](./args-reflect)

A minimal C++ CLI application demonstrating an experimental technique that combines a runtime argument parser with compile-time C++26 reflection to automatically dispatch and convert function parameters.

**Key Features:**
- Automatic parameter parsing and type conversion using C++26 reflection
- Zero-copy string parsing with `std::string_view` and `std::from_chars`
- Support for multiple function dispatching
- Built with clang supporting P2996 reflection
